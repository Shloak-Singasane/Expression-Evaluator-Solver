# Project Topic :Expression-Evaluator
---

## Project Description 
This project builds an expression tree from a postfix expression and then evaluates it using postorder traversal. 
A postfix expression is given as input, and with the help of a stack, the program creates a binary tree where operands become leaf nodes and operators become internal nodes.

Once the tree is built, postorder traversal (left → right → root) is used to evaluate the expression and get the final result. 
This project helps understand how stacks, binary trees, and tree traversals work together to process mathematical expressions.

---

## Data Structures Used
Stack
Binary Tree.

---

## Expected Features
- Accept postfix expression as input
- Build expression tree using stack
- Evaluate expression using postorder traversal
- Display the final result
